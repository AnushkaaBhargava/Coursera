# Coursera Backend API 🎓

A backend application for an online learning platform inspired by Coursera. The project provides APIs for user management, course management, authentication, and role-based access control.

## Features

* User Authentication using JWT
* Admin and User Roles
* Course Creation and Management
* Protected Routes using Middleware
* MongoDB Database Integration
* RESTful API Architecture

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* Zod Validation

## Project Structure

```text
routes/
middleware/
config.js
db.js
index.js
```

## Implemented APIs

* User Signup/Login
* Admin Signup/Login
* Create Course
* Update Course
* View Courses
* Purchase/Enroll in Courses
